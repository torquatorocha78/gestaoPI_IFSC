# gestaoPI_IFSC
📑 Gestão de Patentes – IFSC (Streamlit) gestão de patentes do Instituto Federal de Santa Catarina (IFSC).  O sistema permite cadastrar, consultar, atualizar e acompanhar pedidos de patente, inventores e informações administrativas, com foco no apoio às atividades do NIT / gestão da inovação.
